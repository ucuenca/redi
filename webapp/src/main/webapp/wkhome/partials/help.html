Descripción de REDI

REDI (Repositorio Ecuatoriano De Investigadores) es un sistema que permite identificar áreas comunes de investigación entre una lista de autores y publicaciones de las diferentes universidades del Ecuador. Para lograr este objetivo, se ha definido una serie de paquetes que permitan:

  1. La extracción de los autores por la invocación automática de los servicios que utilizan el estándar OAI-PMH1
  2. La extracción de las publicaciones de los autores de diferentes fuentes de datos
  3. La descripción semántica de los autores y sus publicaciones con respecto a modelos ontológicos
  4. La aplicación de técnicas de minería de datos (clustering) para detectar áreas comunes a través de las palabras clave, título y resumen presentes en las publicaciones.

En este sistema combinamos las ideas de varias iniciativas, y proponemos un nuevo sistema centrado en la identificación de áreas de investigación comunes.

Funcionalidad


Entre las funcionalidades que REDI ofrece se encuentran:

  1. Búsqueda de autores
  2. Navegación a través de las publicaciones del autor
  3. Soporte para consultar publicaciones de autores externos
  4. TAG CLOUD: Nube de etiquetas de áreas de investigación. El tamaño de cada nodo depende del número de publicaciones
  5. GROUP BY: Se muestran las publicaciones según la área de investigación seleccionada y se agrupan por autores
